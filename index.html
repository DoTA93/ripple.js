How to install



Step 1: Download ripplejs files

Download

Step 2: Include the files in your project

<!-- include minified CSS -->
<link rel="stylesheet" type="text/css"  href="ripplejs/dist/ripple.min.css">

<!-- include minified JS -->
<script src="ripplejs/dist/ripple.min.js"></script>

Step 3: HTML markup

<!-- for dark ripple -->
<button class="ripple ripple-dark"></button>

<!-- for light ripple -->
<button class="ripple ripple-light"></button>
